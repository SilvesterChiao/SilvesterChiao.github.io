function gcd(p, q) {
    if (q === 0) {
        return p
    }

    var r = p % q
    return gcd(q, r)
}

# 排序算法

稳定性：排序后 2 个相等键值的顺序和排序之前它们的顺序相同

稳定的排序算法：冒泡排序、插入排序、归并排序、计数排序、基数排序、桶排序

不是稳定的排序算法：选择排序、快速排序、希尔排序、堆排序
