查看网速

```bash
sudo add-apt-repository ppa:fossfreedom/indicator-sysmonitor
sudo apt-get update
sudo apt-get install indicator-sysmonitor
indicator-sysmonitor &
ctrl + c
```

鼠标灵敏度设置

```bash
vim ~/.profile
sudo xset mouse 2.0
```

然后 ctrl + c 后台运行，然后在上图中的显示区域点击图标，然后选择 preference，进行选择开机自启动设置。
